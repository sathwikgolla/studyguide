## ✨ Overview

**PrepFlow** is a structured learning platform for placement preparation.  
It converts scattered preparation into a **clean, trackable, and interactive experience**.

- 📚 Organized modules  
- 📊 Progress tracking  
- 🎯 Topic-based learning  
- 🧠 Notes + revision system  

---

## 🧩 Features

### 📘 Core Modules
- **DSA** → Topic-based (Arrays, Trees, Graphs → Easy/Medium/Hard)
- **OS / DBMS / CN** → Concept-based structured sections
- **Full Stack Development** → Complete roadmap (Frontend, Backend, DB, APIs)
- **Additional Modules** → Aptitude, System Design, DevOps, etc.

---

### ⚡ Smart Functionality
- ✅ Checkbox-based progress tracking  
- 📝 Notes per question/topic  
- 📊 Topic & module progress bars  
- 🔍 Search + filters (topic, difficulty)  
- 🔄 Resume where you left  
- 🎯 Difficulty grouping (Easy / Medium / Hard)

---

### 🎨 UI / UX Highlights
- 🌙 Dark modern theme  
- ✨ Smooth animations (Framer Motion)  
- 🎯 Unique styling per module  
- 📱 Responsive layout  
- 💎 Premium SaaS-style dashboard
