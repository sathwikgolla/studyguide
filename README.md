🚀 PrepFlow – All-in-One Placement Preparation Platform
✨ Overview

PrepFlow is a structured learning platform designed for students preparing for placements.
It transforms scattered preparation into a clean, trackable, and visually engaging experience.

Instead of random practice, PrepFlow gives you:

📚 Organized learning modules
📊 Progress tracking
🎯 Topic-based structured flow
🧠 Notes + revision system
🧩 Features
📘 Core Modules
DSA → Topic-based (Arrays, Trees, Graphs → Easy/Medium/Hard)
OS / DBMS / CN → Concept-based structured learning
Full Stack Development → Complete roadmap (Frontend, Backend, DB, APIs)
Additional Modules → Aptitude, System Design, DevOps, etc.
⚡ Smart Functionality
✅ Checkbox-based progress tracking
📝 Notes per question/topic
📊 Module & topic-wise progress bars
🔍 Search + filter (topic & difficulty)
🔄 Resume where you left
🎯 Difficulty segmentation (Easy / Medium / Hard)
🎨 UI/UX Highlights
🌙 Modern dark theme
✨ Smooth animations (Framer Motion)
🎯 Unique styles per module

🚀 Getting Started
1️⃣ Clone the Repository
git clone https://github.com/your-username/prepflow.git
cd prepflow
2️⃣ Install Dependencies
Frontend
cd frontend
npm install
npm run dev
Backend
cd backend
npm install
npm start
3️⃣ Setup Environment Variables

Create a .env file in backend:

MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
PORT=5000
